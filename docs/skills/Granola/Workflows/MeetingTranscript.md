# MeetingTranscript

Use this workflow when the user wants the transcript for a meeting.

## Primary Command

```bash
granola meeting transcript <meeting-id> --json
```

## Related Commands

```bash
granola meeting <meeting-id> summarize
granola meeting <meeting-id> actions
granola meeting <meeting-id> key-takeaways
```

## Notes

- Transcript is the source of truth when notes are sparse or empty.
- Use the AI helper commands after transcript availability is confirmed.
